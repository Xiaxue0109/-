#### 7. 折半查找：

```c
int binary_search(int *arr, int n, int x) {
    //x 是要查找的数字 n 总个数
    int head = 0, tail = n - 1, mid;//head 头指针，tail 尾指针
    while ( head < tail) {
        mid = (head + tail) >> 1;
        if (arr[mid] == x) return mid;//找到，正好是mid
        if (arr[mid] < x) head = mid + 1; //要找的值大于mid，就将head指针移=mid+1；
        else tail = mid - 1;
    }
    return -1;//没找到
}
```
